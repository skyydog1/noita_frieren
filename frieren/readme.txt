Mod: frieren
Made by: Crusty
__________________________________________________________________

Description:

This mod replaces the player character with Frieren or Fern from Sousou no Frieren, including their respective staffs (staves?).

You can find customization options in the mod settings menu.

Install Guide:
1. Subscribe.
2. Turn on the mod ingame.
3. Start a new game.

Mods load first to last, so whatever mods you want to work hardest should be put first :)
